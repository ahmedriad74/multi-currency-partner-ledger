Sale print forms for Ukraine

This module adds necessary printed forms for Sale transactions in Ukraine.
It includes common PDF reports such as: Rakhunok na oplatu, Vydatkova nakladna, Akt vykonanykh robit.

Settings

Settings / General settings / Companies / Document layouts

A new option has been added to the layout of printed forms as "Light custom". The text entered in the Company Details block will be displayed in the header of the printed form (PDF).

The User should specify the main bank account in the Company's profile which will be used in the PDF printed forms. The available bank accounts listed in the contact associated with the Company can be used for this purpose.

Usage

Through the Print option User is able to create:

from Sale Order the PDF report of Rakhunok na oplatu,

from Customer Invoice the PDF reports of Vydatkova nakladna and Akt vykonanykh robit.
